🎮 UNO Game in Python

A console-based UNO game developed using Python that implements card game mechanics, turn-based gameplay, and computer AI interaction.  
This project demonstrates core programming concepts such as functions, loops, conditional statements, lists, tuples, user input handling, and game logic design.

✨ Features

- 🃏 Complete UNO card deck generation system
- 👤 Single-player gameplay against computer AI
- 🎨 Card matching based on color and value
- 🔄 Implementation of UNO special cards:
  - Skip
  - Reverse
  - Draw Two
  - Wild
  - Wild Draw Four
- 🤖 Automated computer opponent logic
- 📥 Draw card functionality
- 🚪 Exit game option
- 🎲 Randomized deck shuffling using Python Random module

🎯 Game Rules

- Players must match cards based on:
  - Same color
  - Same number/value
  - Wild cards

- Special Action Cards:
  - **Skip** → skips opponent’s turn
  - **Reverse** → reverses gameplay direction (acts as skip in 2-player mode)
  - **Draw Two** → opponent draws 2 cards
  - **Wild** → allows player to choose a new color
  - **Wild Draw Four** → opponent draws 4 cards and player selects new color

- The first player to discard all cards wins the game.
